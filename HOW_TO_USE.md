# 📖 РЪКОВОДСТВО ЗА ИНСТАЛАЦИЯ И РАБОТА

Този документ съдържа подробни инструкции стъпка по стъпка за първоначалното настройване на програмата **CADASTRE** и как да я използвате самостоятелно или съвместно с AutoCAD.

---

## 🛠️ Част 1: Първоначална инсталация и настройка

За да стартирате програмата, трябва да подготвите компютъра си с нужния софтуер.

### Стъпка 1.1: Инсталиране на Python
Ако нямате инсталиран Python:
1. Отидете на официалния сайт: [python.org/downloads](https://www.python.org/downloads/)
2. Изтеглете последната версия за **Windows** (напр. Python 3.11 или по-нова).
3. **ВАЖНО:** По време на инсталацията задължително маркирайте отметката **"Add python.exe to PATH"** (Добавяне на Python към системните пътища). Ако пропуснете това, командата `python` няма да бъде разпозната в конзолата.
4. Завършете инсталацията.

### Стъпка 1.2: Инсталиране на мрежовата библиотека (`requests`)
Скриптът използва външната библиотека `requests`, за да комуникира със сървъра на КАИС.
1. Отворете **Command Prompt** (Терминал) или **PowerShell** в Windows (натиснете `Win + R`, напишете `cmd` и натиснете `Enter`).
2. Изпълнете следната команда:
   ```bash
   pip install requests
   ```
   *Забележка: Можете също да използвате `pip install -r requirements.txt`, ако се намирате в директорията на проекта.*

## 📂 Част 2: Как да използваме програмата

Програмата работи, като чете файл със списък от ЕКАТТЕ кодове (номера на землища), извлича имената и местоположението им от локален речник и автоматично изтегля съответните архивни `.zip` файлове от сървъра на КАИС.

### Стъпка 2.1: Подготовка на входния файл (`selected_ekatte.txt`)
Програмата очаква да намери текстов файл със списък от кодове в папката, от която я стартирате.

1. Създайте текстов файл с име **`selected_ekatte.txt`**.
2. В него въведете ЕКАТТЕ кодовете на населените места, които искате да изтеглите. Всеки код трябва да бъде на **нов ред**.
   *Пример:*
   ```text
   35883
   07332
   07079
   ```
3. Запишете файла.

> ℹ️ *Пояснение на кодовете от примера:*
> * `35883` — гр. Камено
> * `07332` — с. Братя Кунчеви
> * `07079` — гр. Бургас

### Стъпка 2.2: Стартиране на изтеглянето
Можете да стартирате програмата по два начина:

#### Вариант А: Чрез конзола (Command Prompt/PowerShell)
1. Отворете терминал и отидете в папката, където се намира `CAD_IMPORT.py` (използвайки `cd` команда).
2. Стартирайте скрипта с:
   ```bash
   python CAD_IMPORT.py
   ```

#### Вариант Б: Чрез двоен клик (ако е настроена асоциация с Python)
* Просто кликнете два пъти върху файла **`CAD_IMPORT.py`**. Ще се отвори черен прозорец, който ще покаже процеса на изтегляне.

### Стъпка 2.3: Къде отиват свалените файлове?
След приключване на работа, програмата автоматично създава директория **`Изходни/`** в текущата работна папка.
Вътре в нея ще намерите подпапки за всяко землище, кръстени по формата `Име (ЕКАТТЕ код)`:

```
Изходни/
├── гр. Камено (35883)/
│   ├── 35883_поземлени_имоти.zip
│   └── 35883_сгради.zip
├── гр. Бургас (07079)/
│   ├── 07079_поземлени_имоти.zip
│   └── 07079_сгради.zip
```

## 🔌 Част 3: Работа в съвместен режим (AutoCAD / LISP / BAT)

Скриптът е оптимизиран да работи безпроблемно при извикване от външни системи (например от AutoCAD чрез LISP и `.bat` файл):

1. **AutoCAD LISP** записва файл `selected_ekatte.txt` в папката на текущия чертеж.
2. LISP-ът стартира `.bat` файл, който извиква `CAD_IMPORT.py`.
3. `CAD_IMPORT.py` разпознава къде се намира чертежът (работната папка) и записва изтеглените ZIP файлове и папка `Изходни/` **директно при чертежа**.
4. Речникът `ekatte_dict.csv` се зарежда автоматично от папката, в която е инсталиран самият скрипт, без да се налага да го копирате навсякъде.

## 🔄 Част 4: Поддръжка на речника с ЕКАТТЕ кодове

Ако някога локалният файл `ekatte_dict.csv` бъде изтрит или желаете да го обновите с най-новите данни от отворения регистър на населените места в България:

1. Стартирайте помощния скрипт:
   ```bash
   python make_dictionary.py
   ```
2. Скриптът автоматично ще се свърже с базата данни в GitHub, ще свали актуалния списък и ще генерира нов, подреден файл `ekatte_dict.csv` в същата папка.
