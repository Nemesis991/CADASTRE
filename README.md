## ✨ Функционалности

| Функция | Описание |
|---------|----------|
| 🔽 **Пакетно изтегляне** | Автоматично тегли `.zip` файлове с кадастрални данни за множество землища наведнъж |
| 🗂️ **Автоматична организация** | Създава подредена файлова структура по землища в папка `Изходни/` |
| 📖 **Локален речник (ЕКАТТЕ)** | Включен CSV речник с всички 265 населени места в България — автоматично съпоставя ЕКАТТЕ кодове с област, община и име |
| 🔄 **Обновяване на речника** | Скрипт за автоматично генериране/обновяване на речника от открит JSON източник |
| 🛡️ **Устойчивост** | Обработка на мрежови грешки, липсващи данни и невалидни ЕКАТТЕ кодове |
| 📊 **Подробен лог** | Ясни съобщения за статуса на всяка операция (✅ успех / ❌ грешка) |

## 🚀 Използване

### Стъпка 1 — Подготовка на входни данни

Създайте (или редактирайте) файл `selected_ekatte.txt` в **работната директория** (папката, от която стартирате скрипта). Въведете ЕКАТТЕ кодовете на желаните землища — **по един код на ред**:

```text
35883
07332
07079
```

> 💡 **Как да намерите ЕКАТТЕ код?**
> - Проверете файла `ekatte_dict.csv` — съдържа всички налични кодове с имена
> - Или потърсете в [Националния класификатор на ЕКАТТЕ](https://www.nsi.bg/nrnm/index.php?ezession=EKATTE)

### Стъпка 2 — Изтегляне на кадастрални данни

```bash
python CAD_IMPORT.py
```

**Очакван изход:**

```
Работна папка (Чертеж): D:\projects\my_project
Папка на програмата: D:\repo\CADASTRE
Започва автоматично изтегляне на 3 землища...

--- Изтегляне за: гр. Камено (35883) ---
  [V] поземлени имоти.zip - СВАЛЕН
  [V] сгради.zip - СВАЛЕН

--- Изтегляне за: с. Братя Кунчеви (07332) ---
  [V] поземлени имоти.zip - СВАЛЕН
  [V] сгради.zip - СВАЛЕН

--- Изтегляне за: гр. Бургас (07079) ---
  [V] поземлени имоти.zip - СВАЛЕН
  [V] сгради.zip - СВАЛЕН

==================================================
Процесът завърши!
Натисни Enter за затваряне на прозореца...
```

Изтеглените файлове ще бъдат записани в `Изходни/` с подпапки за всяко землище.

## ⚙️ Конфигурация

### Работни директории

Скриптът `CAD_IMPORT.py` работи с **две отделни директории**:

| Директория | Предназначение | Как се определя |
|-----------|---------------|-----------------|
| **Работна папка** (`work_dir`) | Чете `selected_ekatte.txt` и записва в `Изходни/` | Текущата директория (`os.getcwd()`) |
| **Папка на програмата** (`script_dir`) | Чете `ekatte_dict.csv` | Автоматично — папката на `CAD_IMPORT.py` |

Това разделение позволява скриптът да бъде инсталиран на едно място и да се извиква от различни проектни папки.

### Параметри на изтеглянето

В `CAD_IMPORT.py` можете да промените:

```python
# API endpoint на КАИС
base_url = "https://kais.cadastre.bg/bg/OpenData/Download"

# Типове файлове за изтегляне
target_files = ["поземлени имоти.zip", "сгради.zip"]
```

### Пример 3: Интеграция с AutoCAD

Скриптът е проектиран да работи с AutoCAD LISP рутина, която:
1. Генерира `selected_ekatte.txt` с нужните кодове
2. Стартира `CAD_IMPORT.py` чрез BAT файл
3. Кадастралните данни се изтеглят автоматично в папката на чертежа

## 🔄 Обновяване на речника с ЕКАТТЕ кодове

Ако е необходимо да обновите речника с най-актуални данни:

```bash
python make_dictionary.py
```

Скриптът:
1. Изтегля JSON база данни от [GitHub: Places-in-Bulgaria](https://github.com/Kostadin/Places-in-Bulgaria)
2. Парсира данните за всички населени места
3. Генерира нов `ekatte_dict.csv` файл

> ⚠️ **Внимание:** По подразбиране скриптът записва файла в `ekatte_dict.csv` в текущата директория. При необходимост променете пътя в `make_dictionary.py`.

### Формат на речника (`ekatte_dict.csv`)

```csv
ЕКАТТЕ,Област,Община,Населено място
00084,Варна,Аврен,с. Аврен
00151,Бургас,Айтос,гр. Айтос
07079,Бургас,Бургас,гр. Бургас
...
```

│  (ЕКАТТЕ кодове)     │      │  (Локален речник)    │
└──────────┬───────────┘      └──────────┬───────────┘

           ▼                             ▼

    │           CAD_IMPORT.py                  │
    │  1. Чете ЕКАТТЕ кодовете                │

    │  4. Изтегля ZIP файлове от КАИС         │
    └──────────────────┬───────────────────────┘
                       │
                       ▼
    ┌──────────────────────────────────────────┐
    │   https://kais.cadastre.bg/bg/OpenData   │

### Как работи URL конструирането

За ЕКАТТЕ код `35883` (гр. Камено, обл. Бургас, общ. Камено):

```
GET https://kais.cadastre.bg/bg/OpenData/Download
    ?path=област Бургас/община Камено/гр. Камено (35883)/поземлени имоти.zip
```

## 🔧 Отстраняване на проблеми

| Проблем | Причина | Решение |
|---------|---------|---------|
| `ГРЕШКА: Няма намерен файл от AutoCAD` | Липсва `selected_ekatte.txt` | Създайте файла в работната директория |
| `ЕКАТТЕ XXXXX не съществува в речника` | Невалиден или непознат код | Проверете кода в `ekatte_dict.csv` |
| `Не е намерен на сървъра` | Данните не са налични в КАИС | Проверете дали землището има кадастрална карта |
| `Проблем с мрежата` | Мрежова грешка | Проверете интернет връзката и достъпа до `kais.cadastre.bg` |
| `Локалният речник липсва` | `ekatte_dict.csv` не е намерен | Стартирайте `python make_dictionary.py` |
| `UnicodeDecodeError` | Проблем с кодирането | Скриптът автоматично опитва `utf-8-sig` и `utf-8` |
